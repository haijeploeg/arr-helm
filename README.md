# arr-helm

A collection of Helm charts for my homelab. Currently contains the **[piratestack](charts/piratestack/README.md)** chart — a full media stack bundling Sonarr, Radarr, Bazarr, Prowlarr, Sabnzbd, Tautulli and Maintainerr into a single Helm release.

Questions or feature requests? Just open an issue!
